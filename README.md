
## Install

### Using NPM:

```
$ npm i --save react-gf-icons
```

### Using Yarn:

```
$ yarn add react-gf-icons
```

## Usage

### Basic usage

```jsx
import React from 'react'
import { IconAccessibility } from 'react-gf-icons'

export const App = () => (
  <div>
    <IconAccessibility />
  </div>
)
```
